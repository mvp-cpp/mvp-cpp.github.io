下载

推荐普通用户从 GitHub Releases 下载稳定版本；开发者可直接克隆源码仓库进行本地编译。

推荐方式：Releases

适合直接下载已打包版本与查看版本更新说明。

打开 Releases 页面

源码仓库

适合需要自行编译、调试或二次开发的用户。

打开 GitHub 仓库

源码下载与克隆

你可以通过 Git 克隆源码：

git clone https://github.com/mvp-cpp/mvp.git
cd mvp

或者直接在 GitHub 仓库页面下载源码压缩包。

建议的发布内容

  • Windows 可执行程序或压缩包
  • Linux / macOS 对应二进制文件
  • 源码包与版本说明
  • 依赖要求、兼容性说明与更新日志
提示： 安装与编译步骤建议放在 Tutorial 页面 中持续维护。
