Get the source

git clone https://github.com/hyacz/himvp.git
cd himvp

Use the Tutorial page for command-level guidance after cloning or downloading a release package.

Build requirements

  • C++17-compatible compiler
  • CMake environment for source builds
  • Enough CPU, memory and storage for target GWAS workloads
  • Optional CUDA environment when GPU-enabled builds are needed

Release notes and version tracking

  • Check tagged releases before updating production workflows.
  • Keep the exact release tag or commit hash in benchmark and manuscript records.
  • Review compatibility notes when build options or CLI behavior change.

Basic source build

mkdir -p build
cd build
cmake ..
cmake --build . -j

Detailed module options, runtime commands and input-file requirements are documented in Tutorial.

What to publish in each release

  • Tagged source snapshot or source package
  • Platform-specific binaries when they are officially supported
  • Release notes, installation updates and compatibility notes
  • Optional benchmark summaries or performance comparison notes
