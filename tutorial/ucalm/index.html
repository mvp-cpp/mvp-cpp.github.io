Section 5

UCALM

UCALM stands for Unsupervised adaptive Clustering-based Association and Layered Meta integration.

Individual-level Data Workflow

Use --ucalm when starting from genotype and phenotype files. This path covers clustering, subgroup GWAS and layered integration.

Requires: genotype prefix, phenotype file and optional covariates. See Genotype Files and Phenotype File.

himvp --ucalm \
    --bfile demo \
    --pheno demo.phe \
    --pheno-pos 2 \
    --out result/ucalm_full
  • Output: subgroup analysis results plus final UCALM integration output under the selected prefix.
  • Use this path for complex populations when predefined labels are incomplete or too coarse.

Summary-statistics Workflow

Use this path when subgroup or cohort summary statistics already exist and only the UCALM integration stage is needed.

Requires: summary statistics prepared as described in Summary Statistics Input.

himvp --meta \
    --sumstat pop1.sumstat,pop2.sumstat,pop3.sumstat \
    --meta-method 1 \
    --threads 8 \
    --out result/trait1_ucalm_meta
  • Output: UCALM integration results under the selected --out prefix.
  • Use this path only when grouping and subgroup-level GWAS have already been completed.

UCALM Output and Interpretation

UCALM writes method-specific association results under the selected --out prefix. The forward-facing output column name should be treated as UCALM_PVAL.

  • Sort output by the UCALM_PVAL column to identify top markers.
  • Output: association result table plus any subgroup-level files produced by the chosen workflow.
  • Check whether top markers have consistent chromosome, position and allele fields across cohorts.
  • Compare component evidence from FE, MR-MEGA and ACAT to classify likely effect patterns.

Interpretation. Review cohort-level effect direction, sample size and QC before claiming a biologically meaningful UCALM signal.

Practical Notes

  • Use consistent genome builds and allele coding across subgroups or cohorts.
  • Keep clustering results, subgroup sample sizes and subgroup GWAS commands together with final UCALM outputs.
  • Check MAF, sample-size filtering and rare-variant behavior before interpreting UCALM results.
  • Small subgroups can reduce the power of subgroup-specific association tests; avoid over-interpreting signals from underpowered clusters.
  • UCALM can detect statistical heterogeneity, but functional interpretation still requires annotation and biological evidence.
