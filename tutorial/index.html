Section 1

Getting Started

MVP is the modern C++ reimplementation of the rMVP workflow for large-scale GWAS analysis. This guide starts from installation and then walks through data preparation, model execution and downstream visualization.

Overview

The new MVP keeps the core spirit of Memory-efficient, Visualization-enhanced, and Parallel-accelerated, while moving the implementation to C++ for stronger engineering scalability and better control of computational performance.

  • Built for genome-wide association analysis on large cohorts
  • Designed for memory efficiency and multicore execution
  • Organized as a command-line workflow suitable for servers and reproducible pipelines

Installation

Build MVP from source with CMake:

git clone https://github.com/mvp-cpp/mvp.git
cd mvp
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . -j

If your server uses custom BLAS, LAPACK or OpenMP backends, you can extend the build options here with environment-specific flags.

System Requirements

  • C++17-compatible compiler
  • CMake 3.16 or later
  • Linux, macOS or Windows development environment
  • Enough memory and CPU threads for the target GWAS scale
