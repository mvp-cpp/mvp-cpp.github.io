Tutorial

Start from installation, then move to basic analysis and visualization.

Build MVP from source

git clone https://github.com/mvp-cpp/mvp.git
cd mvp
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . -j
