环境要求

  • C++17 或更高版本编译器
  • CMake 3.16 及以上
  • Git
  • Linux / macOS / Windows

安装与编译

推荐通过源码构建：

git clone https://github.com/mvp-cpp/mvp.git
cd mvp
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . -j

如果你的项目依赖第三方库，可以在此处进一步补充依赖安装命令和平台差异说明。

运行示例

下面是示例命令，请按你真实的程序名和参数替换：

常见问题

Q1：访问路径怎么记？

本站已配置为三个固定入口： /home、 /download、 /tutorial。 规范访问形式分别是 /home/、/download/、/tutorial/。

Q2：编译失败怎么办？

请先检查编译器版本、CMake 版本和依赖库是否安装完整，再确认当前系统是否受支持。

Q3：如何提交问题或建议？

请前往 GitHub Issues 进行反馈。
