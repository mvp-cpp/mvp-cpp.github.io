HIMVP

A modern C++ toolkit for scalable GWAS, structure analysis and publication-ready genomic reporting.

HIMVP carries forward the core philosophy of rMVP while rebuilding the workflow in C++ for better portability, cleaner engineering and a more practical path from raw genotype input to association results, structure analysis and figures.

Quick Start

Build locally and connect the docs to current source behavior

The website is maintained in the Pages repository, but the software source of truth lives in hyacz/himvp. Clone the source, build it locally and use the Tutorial page for input requirements, workflow recipes and output interpretation.

  • Clone the current source repository from GitHub.
  • Build himvp in your local environment.
  • Prepare phenotype, genotype and covariate inputs.
  • Run GWAS, inspect PCA or continue into meta-analysis.
git clone https://github.com/hyacz/himvp.git
cd himvp

cmake -S . -B build
cmake --build build -j

./build/himvp --help

Workflow

A compact view of how HIMVP fits into a practical genomic analysis pipeline.

1. Prepare input Load phenotype, covariate and genotype files from PLINK 1 or PLINK 2 style datasets.
2. Run analysis Launch GLM or MLM depending on study design, population structure and model needs.
3. Inspect structure Review PCA, GRM and clustering outputs to understand the genetic background of the cohort.
4. Export figures Summarize with Manhattan, Q-Q and downstream reporting assets for interpretation and publication.
