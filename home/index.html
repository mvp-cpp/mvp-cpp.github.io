Genome-Wide Association Study · Rebuilt from rMVP to C++

MVP brings memory-efficient, visualization-enhanced, and parallel-accelerated GWAS into a modern C++ stack.

MVP is the C++ reimplementation of the original rMVP project. It keeps the core philosophy of handling large-scale GWAS efficiently, while moving toward a more performant, portable, and engineering-oriented software architecture for next-generation genomic analyses.
