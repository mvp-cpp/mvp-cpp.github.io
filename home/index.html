Genome-Wide Association Study · Rebuilt from rMVP to C++

MVP brings memory-efficient, visualization-enhanced, and parallel-accelerated GWAS into a modern C++ stack.

MVP is the C++ reimplementation of the original rMVP project. It keeps the core philosophy of handling large-scale GWAS efficiently, while moving toward a more performant, portable, and engineering-oriented software architecture for next-generation genomic analyses.

Why MVP

From the original rMVP philosophy to a cleaner, faster and more extensible C++ release.

Memory-efficient

Built for large marker matrices and population-scale studies, with an architecture that emphasizes practical efficiency in high-throughput GWAS pipelines.

Visualization-enhanced

Integrates analysis with visual reporting, making it straightforward to inspect phenotype distributions, PCA structure, Manhattan peaks and Q-Q deviations.

Parallel-accelerated

Designed with modern compute in mind, so the C++ implementation can better exploit multicore and performance-oriented workflows across platforms.
